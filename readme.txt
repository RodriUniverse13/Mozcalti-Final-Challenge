Objetivo del Proyecto:
El objetivo principal de este proyecto es poner a prueba los conocimientos adquiridos durante el bootcamp,
demostrando habilidades en Python, Git, patrones de diseño y otras tecnologías relacionadas. A través de
este desafío, buscamos crear una aplicación robusta y bien estructurada, donde implementamos patrones de 
diseño para resolver problemas comunes en el desarrollo de software.
                ____________________________________________________________________

Descripción del Proyecto:
En este proyecto, hemos desarrollado una plataforma de gestión de aprendizaje (LMS). Un sistema que permite 
crear nuevas tareas modularmente, ver dichas asignaciones y mostrar todos sus atributos, y otras funciones las 
cuales detallamos en el index.html que puede ver en la lista de archivos. 

Para abrir el index.html, copiar el repositorio de github a una carpeta y abrir la carpeta "templates", despues
abrir "index.html" en el navegador de su preferencia
______________________________________________________________________________________________________________

Para ejecutar las pruebas, ejecutar el archivo ejecutar_pruebas.py
